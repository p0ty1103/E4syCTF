E4syCTFとは？

E4syCTFは、セキュリティ初心者から中級者を対象にしたオンラインのCTF（Capture The Flag）イベントです。 実践的な問題を通じて脆弱性の理解や攻撃・防御の基礎を身につけることを目的にしています。

初心者フレンドリー

問題は導線が親切に設計されており、初めての方でも取り組みやすい構成です。

オンライン開催

全国どこからでも参加可能。Discordで運営・Q&Aを行います。

多ジャンル

Web / Pwn / Crypto / Revなど、幅広いカテゴリの問題を用意予定です。

学習重視

作問者によるWriteupや解説も後日公開。復習して知識を定着させられます。
